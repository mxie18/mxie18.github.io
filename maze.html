Maze

For this project, we learned Kruskal's Algorithm and applied it to generating unique mazes that only have 1 solution. In addition to creating mazes, the program allows for users to interactively solve the maze, or see an animated solution solved using either BFS or DFS. Some extra quality of life improvements include showing the actual path solution after BFS/DFS is ran and being able toggle visited paths.
